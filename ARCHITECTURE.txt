═══════════════════════════════════════════════════════════════════════════════
                              DATA FLOW EXAMPLE
═══════════════════════════════════════════════════════════════════════════════

1. Claude AI makes request:
   GET https://abc123def456.ngrok-free.app/customer-ltv-by-channel?days=30

2. ngrok receives request:
   • Logs request in dashboard (http://127.0.0.1:4040)
   • Forwards to localhost:3001

3. Express server processes request:
   • Validates parameters (days=30)
   • Calculates date range (last 30 days)
   • Makes API call to Shopify

4. Shopify API responds:
   • Returns order data with customer info
   • Includes pagination links if needed

5. Express server processes data:
   • Groups customers by first purchase channel
   • Calculates LTV, order count, averages
   • Formats response as JSON

6. Response flows back:
   Express → ngrok → Claude AI

7. Claude receives JSON:
   {
     "period_days": 30,
     "by_channel": {
       "app": { "customers": 50, "avg_ltv": 250.00, ... },
       "web": { "customers": 120, "avg_ltv": 180.00, ... },
       "other": { "customers": 10, "avg_ltv": 150.00, ... }
     }
   }

                              SECURITY LAYERS
═══════════════════════════════════════════════════════════════════════════════

Layer 1: ngrok HTTPS
  ✅ All traffic encrypted in transit
  ✅ Valid SSL certificate
  ⚠️ URL is public (anyone with URL can access)

Layer 2: Express Server
  ✅ CORS enabled (allows cross-origin requests)
  ✅ Input validation on query parameters
  ✅ Error handling (doesn't expose sensitive data)

Layer 3: Shopify Authentication
  ✅ API token stored in .env (not exposed)
  ✅ Token never sent to client
  ✅ All Shopify requests authenticated

Layer 4: Shopify API
  ✅ Rate limiting (429 handling)
  ✅ Token-based authentication
  ✅ Scoped permissions (read-only)

  • Single server instance

  • No load balancing

For Production Consider:
  • ngrok paid plan (static domain, no timeout)
