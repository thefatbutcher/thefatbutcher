TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════════

Problem: "ngrok: command not found"
Solution: Use the full path in the command (already provided above)

Problem: "authentication failed"
Solution: Make sure you completed Step 2 with your actual authtoken

Problem: "port already in use"
Solution: The server is already running (this is fine!)

Problem: "401 Unauthorized" from Shopify
Solution: Your Shopify token may need verification. Check:
https://your-store.myshopify.com/admin/settings/apps

Problem: ngrok URL not working
Solution: Make sure ngrok is still running (keep the terminal open)
